Ⅰ. 고체 지구

1. 지권의 변동

4] 판 구조 운동과 마그마 활동

- 마그마 : 지구 내부에서 지각 하부나 맨틀 물질이 녹아서 생성된 용융 물질

* 용융 : 고체가 열에 의해 액체가 되는 현상

* 부분 용융 : 마그마가 생성될 수 있는 환경에서 용융점이 낮은 광물부터 녹아서 마그마 만들어지는 현상


- 마그마의 생성 조건

· 지구 내부 온도 상승에 의한 대륙 지각의 용융

· 압력 감소로 맨틀 용융이 일어나면 맨틀 상승

· 물의 공급에 따른 맨틀 용융 곡선의 하강


- 마그마의 생성 과정

· 현무암질 마그마 (1) : 뜨거운 맨틀 물질이 지표 가까이 상승 -> 압력 빠르게 감소 -> 온도 서서히 낮아짐 -> 부분 용융 발생 -> 마그마 생성

* e.g] 해령, 열점

· 현무암질 마그마(2) : 해양 지각, 해양 퇴적물 섭임 -> 지하 깊은 곳에 도달 -> 온도, 압력 증가 -> 퇴적물과 지각의 함수 광물에 포함된 물 방출 -> 맨틀에 물 공급 -> 맨틀의 용융점 낮춤 -> 맨틀의 부분 용융 -> 마그마 생성

* e.g] 섭입대

* 함수 광물 : 화학 결합 내부에 수산화 이온(OH-)을 포함하고 있는 광물. 가열하면 물이(H20)이 빠져나옴

· 유문암질 마그마 : 생성된 현무암질 마그마가 상승 -> 대륙 지각 도달 -> 대륙 지각의 온도 상승 ->


- 화성암의 종류

· 화성암 : 생성 위치와 조성에 따라 현무암질· 안산암질· 유문암질 마그마로 구분되며, 이러한마그마가 식어서 굳어진 암석

* 마그마가 냉각되는 속도에 따라 화산암과 심성암으로 분류됨

· 화산암 : 마그마가 지표 부근으로 분출하여 용암이 빠르게 냉각되면서 굳어져 만들어짐에 따라 구성 광물의 입자 크기가 작음

· 심성암 : 지하 깊은 곳에서 마그마의 관입으로 서서히 냉각되면서 굳어져 만들어짐에 따라 구성 광물의 입자 크기가 큼

Ⅰ. 고체 지구

2. 지구의 역사

- 지질 시대 : 약 46억 년 전 지구가 생기고 지각이 형성된 후부터 현재까지의 시대


- 분류

(1) 고생대 : 하등 동물이나 원시 어류, 양서류와 같은 생물들이 화석으로 산출됨

(2) 중생대 : 거대한 파충류나 암모나이트와 같은 생물들이 화석으로 산출됨

(3) 신생대 : 포유류와 같이 현존하는 생물들과 비슷한 생물들이 화석으로 산출됨
